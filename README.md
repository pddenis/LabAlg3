# Задача 1 — Система обработки задач (Queue)

## Описание

Реализация очереди задач на языке Python с интерактивным консольным меню.  
Очередь работает по принципу **FIFO** (First In — First Out): первый добавленный элемент извлекается первым.

---

## Список задач

### Обязательная часть

 `enqueue(task)` — добавить задачу в очередь
 `dequeue()` — извлечь первую задачу
 `front()` — посмотреть первую задачу без удаления
 `is_empty()` — проверить, пуста ли очередь

### Вариативная часть

Вывод очереди в виде строки

---

## Реализация

### Класс `Task`

Представляет одну задачу с тремя полями.

```python
class Task:
    def __init__(self, task_id, description, priority):
        self.id = task_id
        self.description = description
        self.priority = priority

    def __str__(self):
        return f"id={self.id}, описание='{self.description}', приоритет={self.priority}"
```

`__str__` нужен для удобного вывода задачи через `print()`.  
Без него `print(task)` вывел бы что-то вроде `<__main__.Task object at 0x...>`.

---

### Класс `TaskQueue`

Хранит задачи в списке `self.items`. Все операции работают с началом и концом этого списка.

```python
class TaskQueue:
    def __init__(self):
        self.items = []
```

---

#### `enqueue(task)` — добавление задачи

```python
def enqueue(self, task):
    self.items.append(task)
```

Новая задача добавляется в **конец** списка. Это соответствует принципу очереди — новый элемент встаёт последним.

---

#### `dequeue()` — извлечение задачи

```python
def dequeue(self):
    if self.is_empty():
        print("Очередь пуста")
        return None
    return self.items.pop(0)
```

Удаляет и возвращает элемент с **нулевого индекса** — то есть первый добавленный.  
Перед этим проверяет, не пуста ли очередь, чтобы не получить ошибку.

---

#### `front()` — просмотр первой задачи

```python
def front(self):
    if self.is_empty():
        print("Очередь пуста")
        return None
    return self.items[0]
```

В отличие от `dequeue()`, задача **не удаляется** — только возвращается для просмотра.

---

#### `is_empty()` — проверка на пустоту

```python
def is_empty(self):
    return len(self.items) == 0
```

Возвращает `True`, если в очереди нет ни одной задачи. Используется внутри `dequeue()` и `front()`.

---

#### `__str__()` — вывод очереди строкой (вариативная часть, п. 10)

```python
def __str__(self):
    if self.is_empty():
        return "Очередь пуста"
    parts = " -> ".join(f"[{task}]" for task in self.items)
    return f"Очередь: {parts}"
```

Собирает все задачи в одну строку, разделяя их стрелкой `->`.  
Стрелка наглядно показывает порядок элементов — слева первый, справа последний.

Пример вывода:
```
Очередь: [id=1, описание='Починить баг', приоритет=2] -> [id=2, описание='Написать тесты', приоритет=1]
```
