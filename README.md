# Система обработки задач (Queue)

---

## Постановка задачи

Реализовать очередь задач.

Каждая задача содержит:

* **ID** — идентификатор задачи;
* **Описание** — текстовое описание задачи;
* **Приоритет** — уровень приоритета задачи.

### Обязательная часть

Реализовать следующие методы:

* `enqueue(task)` — добавление задачи в очередь;
* `dequeue()` — удаление первой задачи из очереди;
* `front()` — получение первой задачи без удаления;
* `isEmpty()` — проверка очереди на пустоту.

### Вариативная часть №10

Реализовать вывод содержимого очереди в виде строки.

---

# Структура программы

## Структура Task

Структура `Task` предназначена для хранения информации о задаче:

* идентификатор;
* описание;
* приоритет.

## Класс TaskQueue

Класс `TaskQueue` управляет очередью задач и содержит основные методы работы с ней.

| Метод        | Назначение                  |
| ------------ | --------------------------- |
| `enqueue()`  | Добавление задачи в очередь |
| `dequeue()`  | Удаление первой задачи      |
| `front()`    | Получение первой задачи     |
| `isEmpty()`  | Проверка очереди на пустоту |
| `toString()` | Вывод очереди в виде строки |

# Проверка корректности ввода

Для предотвращения ошибок при вводе данных реализована функция:

```cpp
int inputNumber(string message)
```

Функция:

* выводит сообщение пользователю;
* проверяет, что введено число;
* обрабатывает ошибочный ввод;
* запрашивает повторный ввод до получения корректного значения.

Это позволяет избежать зависания программы и бесконечных циклов при вводе букв вместо чисел.
